Install LiteIDE


Install golang, if needed

http://golang.org/doc/install.html


Install LiteIDE


Install windows

    * Download liteide-win32-bin-0.1.6.zip and extract to c:\ 

    * Load liteide on gui, If golang root is c:\go

      c:\liteide\bin\liteide.exe

    * Load liteide on cmd

      >set GOROOT=c:\go
      >set GOARCH=386
      >set GOOS=windows
      >cd\liteide\bin
      >liteide.exe

Install Linux x86_64

    * Download liteide-linux-x86_64-bin-0.1.6.tar.bz2 and extract to $HOME/
    * Download qtlib-linux-x86-64-4.7.0.tar.bz2 and extract $HOME/liteide/bin 

    * Load liteide on gui, If golang root is $HOME/go

      HOME/liteide/bin/liteide.sh

    * Load liteide on shell

      $export GOROOT=$HOME/go
      $export GOARCH=amd64
      $export GOOS=linux
      $cd/liteide/bin
      $liteide.sh

Install Source

    * Install QT4.7, if needed
    * Fetch the repository

      hg clone https://liteide.googlecode.com/hg/ liteide 

    * Install gopromake

      $ cd liteide/src/go/promake
      $ gomake

    * Install goastview

      $ cd liteide/src/go/astview
      $ gopromake -gopro astview.pro

    * Install liteide

      $ cd liteide/src
      $ qmake 
      $ make release
